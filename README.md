# node-generator-erest

Yoman generator for [erest](https://github.com/yourtion/node-erest). Easy build rest api

## Intsall

```bash
$ npm install -g yo
$ npm install generator-erest -g
```

## With Express

```bash
$ yo erest:express
```

## With @leizm/web

```bash
$ yo erest:lei-web
```
